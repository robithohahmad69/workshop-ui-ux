# BABA FARM SURABAYA

Website untuk BABA FARM - Penyedia benih, pakan, dan perlengkapan budidaya ikan lele berkualitas terbaik di Indonesia.

## 🌐 Live Website

[https://baba-farm.vercel.app](https://baba-farm.vercel.app)

## 📁 Project Structure

```
uas/
├── pelanggan/          # Customer-facing pages
│   ├── index.html      # Homepage
│   ├── katalog.html   # Product catalog
│   ├── detail-produk.html
│   ├── artikel.html    # Blog articles
│   ├── keranjang.html  # Shopping cart
│   ├── pembayaran.html
│   └── history.html    # Transaction history
├── admin/              # Admin dashboard
│   ├── dashboard.html
│   ├── produk.html
│   ├── transaksi.html
│   ├── artikel.html
│   ├── pengguna.html
│   ├── profile.html
│   └── settings.html
├── css/               # Stylesheets
├── js/                # JavaScript files
└── vercel.json        # Vercel deployment config

```

## 🚀 Deployment

Project ini di-deploy menggunakan [Vercel](https://vercel.com).

### Deploy Manual

```bash
# Build & deploy ke Vercel
vercel --prod
```

## 📦 Fitur

- **Pelanggan**: Browse produk, artikel, keranjang belanja, riwayat transaksi
- **Admin**: Dashboard, manajemen produk, transaksi, artikel, pengguna
- **Responsive**: Desktop & mobile friendly
- **Glass Morphism Design**: Modern UI dengan efek glass

## 🛠️ Teknologi

- HTML5
- CSS3 (Glass Morphism design)
- Vanilla JavaScript
- Font Awesome icons
- Google Fonts (Inter, Poppins)
